## Docs Setup

### Page name

_config.yml

```yml
title: Some Title
```

### Index

`index.md` - main english index

Main content is list of links to md files.  
Link to dev-notes page at the bottom.  

For example:

```md
## Content

- []()

---

[Dev Notes](https://atari-monk.github.io/dev-notes/)

---
```

### Structure

- `docs/en` - folder with english content
- `docs/project.md` - file to define what this project is and what is not
- `docs/en/*.md` - any md note with topic related to project
